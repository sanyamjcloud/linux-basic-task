#Linux environment Setup & Exploration Task 1

##Overview
This repository contains my work for task 1 of the internship, focused on seting up an
ubuntu linux environment and exploring basic linux commands.

##What I did
-Installed ubuntu linux on virtualbox
-Explored the linux directory structure using commandsd like pwd, ls, cd
-Created and deleted directories and files using mkdir, touch, rm and rmdir
-Practiced file viewing and editing using cat, less and nano
-Learned file permissions and ownership using chmod and chown
-Performed system monitoring with top, df, -h and free -m
-Documented all commands and outputs in markdown files

##Files included
-'command log.md'
-'summary.md'
-'screenshots/'

##Outcome
By completing this task, I gained confidence in navigating and managing Linux
systems via the command line.
